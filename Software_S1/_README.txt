Software S1

Source files (in ZMX and ZPL format) for conducting the optical simulations using the non-sequential mode in Ansys Zemax OpticStudio 2024 R2.

To get Software S1 working, 
	1. WormScat_macro.ZPL should be copied to the Macros folder sourced by Zemax. 
		On a Windows PC, it will be: C:\Users\...\Documents\Zemax\Macros
		Edit the WormScat_macro.ZPL according to its instructions.
	2. All the STEP files in Design File S1 should be copied to the CAD files folder sourced by Zemax.
		On a Windows PC, it will be: C:\Users\...\Documents\Zemax\Objects\CAD Files
